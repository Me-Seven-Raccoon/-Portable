R (programming language)

1. Способ распространения Бесплатно
2. Описание Это язык с рабочим приложением, которое является стандартом для анализа данных. Лучше всего подходит для продвинутых пользователей, которым необходим мощный инструмент. Функциональные возможности: сбор данных.
3. Взаимодействие с программой Введение кода
4. Ссылка на официальный сайт https://www.r-project.org/

RStudio

1. Способ распространения Бесплатно
2. Описание RStudio ™ - это интегрированная среда разработки (IDE) для R. RStudio объединяет инструменты для работы с R в единой настраиваемой среде. Функциональные возможности: подсветка синтаксиса; управление сессиями; IDE; встроенный отладчик; завершение кода.
3. Взаимодействие с программой Введение кода
4. Ссылка на официальный сайт https://www.rstudio.com/

PSPP

1. Способ распространения Бесплатно
2. Описание PSPP - это программное приложение для анализа выборочных данных. Он имеет графический интерфейс пользователя и обычный интерфейс командной строки. Написан на C, использует научную библиотеку GNU для своих математических процедур и plotutils для генерации графов. Предназначается для бесплатной замены проприетарной программы SPSS.
3. Взаимодействие с программой Графический интерфейс
4. Ссылка на официальный сайт https://www.gnu.org/software/pspp/

IBM SPSS Statistics

1. Способ распространения Платно
2. Описание Программная платформа IBM SPSS предлагает расширенный статистический анализ, обширную библиотеку алгоритмов машинного обучения, анализ текста, расширяемость с открытым исходным кодом, интеграцию с большими данными и плавное развертывание в приложениях. Функциональные возможности: импорт из Excel.
3. Взаимодействие с программой Графический интерфейс

SOFA Statistics

1. Способ распространения Бесплатно
2. Описание SOFA Statistics - это статистический пакет, в котором особое внимание уделяется простоте использования, обучению по ходу работы.Он может напрямую подключаться к MySQL, SQLite, MS Access и MS SQL Server. Данные также могут быть импортированы из файлов CSV или MS Excel. Анализ и отчетность могут быть автоматизированы с использованием скриптов Python - либо экспортированы напрямую из SOFA Statistics, либо написаны вручную.
3. Взаимодействие с программой Графический интерфейс
4. Ссылка на официальный сайт http://www.sofastatistics.com/home.php

GMDH Shell

1. Способ распространения Платно
2. Описание Программное обеспечение для прогнозирования. GMDH Shell - это один из способов прогнозировать временные ряды, создавать классификаторы и регрессионные модели. Основанный на искусственных нейронных сетях, он позволяет создавать прогностические модели. Функциональные возможности: прогнозирующее моделирование; сбор данных; аналитика данных.
