Коэффициент корреляции Пирсона

1.Тип решаемой задачи Поиск взаимосвязи переменных
2. Для чего используется Критерий корреляции Пирсона позволяет определить, какова теснота (или сила) корреляционной связи между двумя показателями, измеренными в количественной шкале. При помощи дополнительных расчетов можно также определить, насколько статистически значима выявленная связь.
3.В каких случаях можно использовать Переменная выражена в количественной шкале; анализируемые величины должны быть распределены по нормальному закону
4.Нулевая гипотеза(Н0) Отсутствкет линейная связь между переменными; альтернативная гипотеза Н1 имеется линейная связь между переменными
5.Интерпретация При р<0,05 корреляция между переменными не значима; при р>0,05 корреляция между переменными значима
6.Может быть расчитан в программах: Statistica; Excel; MATLAB; R-Studio; SPSS; GraphPad; Statgraphics

Критерий Колмогорова-Смирнова:

1.Тип решаемой задачи Проверка соответствия эмпирического распределения исследуемой величины нормальному закону.
2. Для чего используется Данный критерий позволяет оценить соответствие распределения исследуемой выборки выраженной в количественной шкале закону нормального распределения.
3.В каких случаях можно использовать Исследуемая величина должна быть выражена в количественной шкале; объём выборки должен превышать 50 наблюдений, при объёме выборки от 25 до 50 эллементов целесообразно применение поправки Большево; Критерий Колмагорова-Смирнова подходит только для проверки соответсвия эмпирического распределения нормального закона с заранее заданными характеристиками; в случае, если характеристики нормального закона распределения требуется получить из экспериментальных данных, необходимо использовать критерий Лиллиефорса.
4.Нулевая гипотеза Эмпирическое распределение соответствует нормальному закону, альтернативная не соответствует
5.Интерпретация Если значение критерия значимо, то гипотеза, что соответствующее распределение нормальное, должна быть отвергнуто
6.Может быть расчитан в программах: Statistica; SPSS; R-Studio
